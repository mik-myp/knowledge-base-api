## 3. 当前最小 ask 接口定义

### 请求

```json
{
  "knowledgeBaseId": "507f1f77bcf86cd799439013",
  "sessionId": "可选，继续已有会话时传入",
  "question": "这个知识库里是如何设计鉴权的？"
}
```

### 响应

```json
{
  "sessionId": "xxx",
  "answer": "......",
  "sources": [
    {
      "documentId": "xxx",
      "documentName": "鉴权设计.md",
      "chunkSequence": 0,
      "page": 2,
      "startIndex": 320,
      "endIndex": 468
    }
  ]
}
```

### 当前最小 LangChain 持久化规则

- `HumanMessage` -> `chat_messages(messageType=human, content=question)`
- `AIMessage` -> `chat_messages(messageType=ai, content=answer, sources=...)`
- `chat_sessions` 只负责承接 `sessionId`
