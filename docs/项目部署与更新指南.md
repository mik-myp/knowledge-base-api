# 项目部署与更新指南

## 1. 当前启动目标

先把已有代码跑起来：

- 用户登录
- 知识库管理
- 文档上传或编辑器录入

`document_chunks`、`chat_sessions`、`chat_messages` 和 `chat/ask` 当前还没有在代码里实现。

---

## 2. 当前环境变量

最小必需：

```env
MONGODB_URI=mongodb://localhost:27017/knowledge-base-api
JWT_ACCESS_SECRET=knowledge-base-api-access
JWT_REFRESH_SECRET=knowledge-base-api-refresh
```

对象存储可选：

```env
AWS_EMD_POINT=https://your-object-storage-endpoint
AWS_ACCESS_KEY_ID=your-access-key
AWS_SECRET_ACCESS_KEY=your-secret-key
S3_BUCKET_NAME=your-bucket
```

如果对象存储未配置：

- 仍可先用编辑器文档继续开发
- 也可上传 `md` / `txt`
- 不建议上传 `pdf` / `doc` / `docx`

---

## 3. 当前验证命令

```bash
pnpm build
pnpm test -- users knowledge_bases documents
```
